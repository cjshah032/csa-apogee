Instructions
This is a Data preprocessing contest. You are given raw data and shall submit 2 csv files (training set and test set).
  • The target attribute is Cover_Type.
  • The file train.csv contains data with target value i.e. Cover_Type while test.csv has no target value.
    • Shape of train.csv = (50000, 56)
    • Shape of test.csv = (10000, 55)
  • [IMPORTANT] Participants have to implement the same pre-processing methods with both train.csv and test.csv separately. Otherwise, accuracy would drop greatly.
  • The submitted test.csv must have the same number of rows in the same order as in the provided test.csv. Deletion of any row or swapping would cost huge.
  • An ML model shall be trained on the preprocessed data that you have submitted and its performance on the test set (which you have preprocessed with the training set and submitted) will be used to rank participants.
  • The model used for the classification will be SVC (Support Vector Classifier).
    • Support Vectors Classifier tries to find the best hyperplane to separate the different classes by maximizing the distance between sample points and the hyperplane.
    • Given model has kernel as ‘poly’ with degree 3 and penalty C as 1.
    • [Tip] You are free to try this out on your own and see how it performs on your data before making your submission.
  • F1 score is used with average as ‘weighted’ for evaluating the accuracy of the model.
    • F1 = 2 * (precision * recall) / (precision + recall)
  • Submitted files must not contain any missing or object type value if any score of 0 will be awarded to the participant.
